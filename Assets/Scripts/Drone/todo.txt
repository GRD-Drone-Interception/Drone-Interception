==========[DRONE TYPE SELECTION]==========
Click on DroneType button
    Instantiate a Drone at the podiums position
    
==========[DRONE COMPONENT SELECTION]==========   
Click on Component button
    Instantiate a component that follows the mouse cursors position 
    If the mouse is clicked AND mouse is in the bounds of an AttachmentPoint
        Parent the component to the AttachmentPoint
        Set the position of the component to that of the AttachmentPoint
    else
        Destroy the currently selected component
        
==========[PODIUM MOVEMENT]==========
Parent drone to podium
Rotate podium using A and D
